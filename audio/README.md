# Pasta de Áudio

## Como adicionar a música "Ordinary"

1. Baixe a música "Ordinary" em formato MP3
2. Renomeie o arquivo para `ordinary.mp3`
3. Coloque o arquivo nesta pasta (`audio/ordinary.mp3`)

## Fontes para baixar música:

- **YouTube to MP3**: Use um conversor online como:
  - https://ytmp3.cc/
  - https://www.y2mate.com/
  - https://320ytmp3.com/

- **Spotify**: Se você tem a música no Spotify, pode usar conversores específicos

- **SoundCloud**: Se disponível no SoundCloud

## Formato recomendado:
- **Formato**: MP3
- **Nome do arquivo**: `ordinary.mp3` (exatamente assim, em minúsculas)
- **Qualidade**: 192kbps ou superior para melhor qualidade de áudio

## Importante:
⚠️ Certifique-se de que você tem os direitos para usar a música ou que está usando de acordo com as leis de direitos autorais do seu país.
